# 📧 Email Security Scanner 🔐  This is a Python-based tool that analyzes email content to detect phishing attempts.  ## 🚀 Features - Detects phishing keywords - Identifies suspicious links - Checks fake domains - Provides risk level (Low / Medium / High) - Gives security tips  ## 🛠 Technologies Used - Python - Regex - URL Parsing  .
